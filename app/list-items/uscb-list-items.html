Description

Purpose

List items provide the user with a link to view more information.

Usage

List items consist of a text link, a dropdown, table ID and buttons to take further action on the data associated with the list item. Users can select a list item to view information about a particular statistic in a table or map view. Results are shown as list items in CEDSCI, and have different formats depending on the current page view (detailed below).

  • All Results
    • Text Link, Dropdown, Table ID, View Table Button, Download Button
  • Table Results
    • Text Link, Dropdown, Table ID, View Table Button, View Thematic Map Button, Download Button
  • Web Page Results
    • Text Link, URL
  • Map Results
    • Text Link, Dropdown, Table ID, Download Button
Do
  • Allow the user to click or tap anywhere on the whole line item to take them to the corresponding link
  • Highlight the line item when the user hovers over it to indicate the clickable area

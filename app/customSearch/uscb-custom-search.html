Description

Purpose

Search widgets are means for users to look for content without manually clicking through the web page.

Usage

Search widgets can take on a variety of aesthetic qualities, but they essentially serve similar purposes: to find content without having to dive into web pages using many clicks. These widgets are most commonly known as "Search Bars," and include an area for text input and a call to action that initiates the search. Conventional search bars also include predictive text, which completes users' search terms using character matching algorithms.

Types
  • Single Search bar
  • Results Search
  • Filter Search
  • Maps tool Search
Do
  • Provide predictive text to help users find what they need.
  • Differentiate different kinds of queries in the predictive texts by section (tables, keywords, etc.).
  • Take advantage of both global search bar patterns and local search bar patterns.
Do not
  • Overcomplicate the search bar.
  • Make too many search bars.

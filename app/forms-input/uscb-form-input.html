Description - Text Input

Purpose

Text input fields allow the user to type a specific query.

Usage

Text input fields remove hint text and show a cursor when the user clicks into the field. Users can enter a combination of letters and numbers. Text inputs are use primarily as search inputs in the CEDSCI platform. 

Do
  • Remove hint text as soon as user clicks into text field 
  • Focus on text box when user clicks into text field
Do not

Description - Select

Purpose

Dropdowns allow users to make a selection from a predefined set of choices.

Usage

On click of a dropdown box, the user can select an item from a dropdown list. List items might result in different information displayed to the user. Dropdowns can also be used to select an option from a list of mutually exclusive options. Dropdowns include an area for displaying content and an indication that there are more options available, such as an arrow sign. E.g. sort dropdown. 

  • Dataset selection inside Table result list items – Allows user to select a specific survey year.
  • Dataset selection inside Table expand version – Allows user to select a specific survey year.
  • "Sort by" dropdown: rearranges result list items based on selected parameter.
  • "Map layers to show" selection overlay menu on Map tool – Allows user to select Boundary and Label to show on selection map.
  • Geography type dropdown in Map tool – Allows user to select specific geography types to show on map.
Do
  • Allow user to select one item from dropdown at a time
  • Display selected dropdown item in default/collapsed dropdown view
Do not
  • Allow user to select more than one item from dropdown list
  • Allow to the dropdown to spill over the viewport

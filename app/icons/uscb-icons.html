Description

Purpose

Icons are pictorial representations of content labels. Icons are used as a way to abbreviate content labels through pictorial means.

Usage

In a responsive or adaptive webpage, these icons serve as buttons. Icons as buttons perform one function across any given experience or platform. There are multiple occurrences throughout the CEDSCI platform: results instances (Table/Map/Download icons, expand view, Share), the Filter side panel (collapsed state), and any Close icons ("x").

Do
  • Provide consistent usage across platform, ensuring that icons only perform one intended action with interaction.
  • Make icons large enough to be visually discernable for most users (minimum 15 x 15px).
  • Use labels for common user actions.
Do not
  • Use icons as complete replacement for labels in full desktop viewport, unless they are a part of a collapsed menu (expand on mouse hover).
  • Include icons on every label.

Code

<div> <p>Close</p> <i class="fa fa-close"></i> </div> <div> <p>Map</p> <i class="fa fa-globe"></i> </div> <div> <p>Download</p> <i class="fa fa-download"></i> </div> <div> <p>Share</p> <i class="fa fa-share-alt"></i> </div> <div> <p>Table</p> <i class="fa fa-table"></i> </div> <div> <p>Search</p> <i class="fa fa-search"></i> </div> <div> <p>Expand</p> <i class="fa fa-expand"></i> </div>

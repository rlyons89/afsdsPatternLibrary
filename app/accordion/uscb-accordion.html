Description

Purpose

Accordions are used to present large bodies of content in a condensed, hierarchical manner, with children expanding upon user action.Â 

Usage

Accordions present content through click-through hierarchies. Parent content labels are displayed first, and subsequent levels of children content expand when the user interacts with the parent content's label. The search "Filters" side panel is the only occurrence of accordion in the platform. Topics, Geographies, Years, Survey/Dataset, and Industry (NAICS) are the first level content items in the Filter side panel. Child content items use a different color to indicate placement under parent content, and have checkbox interaction patterns.

Types
  • Search "Filters" side panel.
Do
  • Use hover states on accordion items to reflect interactivity.
  • Include option for the user to search items instead of clicking through to find, if the accordion has many content items.
  • Allow for a scroll interaction if the accordion becomes too large for the screen.
Do not
  • Bury content in more than 3 layers or hierarchy. If content is under 3 levels, use other means of displaying content items.

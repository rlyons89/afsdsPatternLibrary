Description

Purpose

Toasts provide feedback about an action taken by the user on the current page. 

Usage

Toasts appear  when a certain action has been taken by the user to provide feedback. Toasts appear in the top right corner of the screen and disappear after 4 seconds. In CEDSCI, toasts are used on the Map page.

Do
  • Keep the length of the message short
  • Keep the placement toasts on the top right of the page
Do not
  • Change the placement of toasts on the page
  • Allow the user to exit out of toasts

Code

Example

Error Toast Warning Toast Success Toast Info Toast

Description

Purpose

Progress indicators convey the current system state to the user, with the aim being to level the user's expectations during loading.

Usage

Progress indicators overlay the screen or screen elements. Current implementation uses a consistent radial load indicator; the only occurrences in the current platform are whenever the map component and table components are loading.

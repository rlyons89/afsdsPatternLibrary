Description

Full Width Content
 

This can be used for general layouts (ex: landing page)

 
Full-Width Content with Side Nav

This can be used for layouts with the filter bar collapsed on load of page and one type of content spanning the whole page.  

50-50 Content with Side Nav

This can be used for layouts with filter bar expanded on load of page. Search results and content on right side use equal amounts of space on page (ex: Tables results). 

30-70 Content with Side Nav

This can be used for layouts with filter bar collapsed on load of page, search results taking up 30% of page and content taking up 70% of page (ex: Maps results).

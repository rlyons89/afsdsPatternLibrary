Description

Purpose

Pagination shows the user that more information is available, beyond what is shown on the current screen.

Usage

Pagination uses numbers to indicate the user's current position among available pages, and a spatial arrangement of numbers to show further available information.

The CEDSCI platform leverages pagination on the results pages, in the event that there are more than 10 results. In the expanded table components, pagination elements are present regardless of the number of rows and columns can fit in the current view; they become active in the event that there are more than 20 rows , while the number of columns depends on the physical width of the content.

Do
  • Use numbers to indicate current page numbers.
  • Indicate the total number of pages.
  • Separate page numbers using minimal graphical elements, such as light opacity lines.
Do not
  • Use excessive space to separate page numbers.

Description

Purpose

Checkboxes are used as indication that an object can be selected from a visible list of one or more options.

Usage

Checkboxes have two different states: unselected and selected. The selected state is indicated by a change in the appearance of the box. Checkboxes can be used for lists of more than 1 option that are non-mutually exclusive. CEDSCI uses checkboxes in the Filter panel on individual filter items as well as the "Transpose" feature on table expanded views.

Do
  • Make a larger area than the physical box correspond with triggering the selected state, such as the label that the box corresponds to.
  • Size the box to web accessibility standard.
Do not
  • Make only the physical box on checkbox entity interactive.

Code

<md-checkbox md-no-ink aria-label="Checkbox No Ink" class="uscb-primary-checkbox"> Checkbox (uscb-primary-blue): No Ink </md-checkbox> <md-checkbox aria-label="Checkbox No Ink" class="uscb-primary-checkbox"> Checkbox (uscb-primary-blue) </md-checkbox>

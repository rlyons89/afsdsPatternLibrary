Description

Purpose

Buttons are used as triggers for actions. Buttons communicate the possible outcome as a result of the user action.

Usage

They come in a variety of shapes, sizes, and colors; they also contain varying amounts of text and/or iconography.

  • Major call to action. These buttons can be found on the landing page "Start Exploring" in a higher contrast orange color and on the Single search page in a darker blue. These buttons are major calls to action because they draw attention through ample use of whitespace around them and color differentiation.
  • Iconographic buttons are simple calls to action. These are intended to convey the destination of the link through symbolic representation of the content without visually overstimulating the user. These are used in: Search button (indicates a new search event), view Map, download, share, minimize/close, and any controls on the Map pages.
  • Text button: buttons can also assume a text-label form, in which there is no border or physical indication that the area is interactive until the user hovers over the area. The top navigation in is an example of this. View more details on text links.
Do
  • Make only the most important actions noticeable with primary buttons.
  • Adhere to a reduced color palette.
  • Use hover states to indicate interactivity.
Do not
  • Make every button a similar color.
  • Use high contrast colors on non-primary buttons.
  • Use the button as the only way to convey actions: provide supporting text where necessary.

Description - Text Links

Purpose

Links are used to navigate the user either between pages and views within the application or to external sites.

Usage

Text links appear in blue and become underlined on hover. On click, the user can navigate to the corresponding page. Links should have title attribute associated with them. There are two kinds of links in the CEDSCI platform: URL hyperlinks and semantic text links. URL hyperlinks include the feedback email address, webpage results URLs, and a link back to the census.gov data. Semantic text links are used ubiquitously throughout the interface.

Do
  • Keep links below one sentence long.
  • Ensure that the verbiage used on the link produces an easily expected outcome.
  • Make all displayed URLs into links.
  • Alter the visual state of links once clicked.
  • 1-to-1 relationship between any link and destination.
Do Not
  • Use too different kinds of hover states for similar styles of links.
  • Make links more than 3 or 4 words long.

Description

Purpose

Allows the user to switch between multiple focused panes/views of content. 

Usage

Tabs manage sets of content, giving the users the ability to view only the content pane they are interested in. Each tab has a corresponding content pane with list items. Users can select a tab to navigate between different kinds of content. In CEDSCI, tabs are used as persistent navigation elements on the results pages: All (displays table and webpage results), Tables (displays only table results), Web Pages (displays a list of relevant webpages), and Maps (an interactive selection map that displays table results as thematic maps, when paired with a geography).

Do
  • Use visual indication for selected tab, such as a small bar underneath the tab item and change in text color 
  • Highlight the line item when the user hovers over it to indicate the clickable area
Do not
  • Allow for selection of more than one tab at a time

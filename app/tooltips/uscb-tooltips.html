Description

Purpose

Tooltips provide the user with a description of an element on the page.Â 

Usage

Tooltips appear when the user hovers over an element and disappears when the user moves their cursor away from the element. The CEDSCI platform uses a tooltip on the primary landing page in the "Phased Releases" section, where the user can hover over various parts of the graphical timeline and learn more about each released of the CEDSCI platform. The intent here is to provide more information in a visually discreet manner.

Do
  • Use consistent iconography to indicate a tooltip.
  • Make tooltips short, digestible bits of information about the appropriate widget.
Do not
  • Redirect the user when they click on the tooltip, unless there is a link within the text.

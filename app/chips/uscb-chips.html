Description

Purpose

Chips allow the user to know which search parameters they have selected.

Usage

As the user selects filters from the accordion (Filter side panel), chips appear below the tabs to indicate to the user which filter topics have been selected. Chips can be removed by selecting the "x" on each chip. The "x" on each chip changes colors on hover to indicate that it is clickable.

Do not
  • Create a horizontal scroll to view more chips than those that fit across the screen on one line
  • Abbreviate or truncate text on each chip
